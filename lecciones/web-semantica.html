La Web Semántica

La Web Semántica fue una iniciativa del W3C creada en 2001 con el objetivo de que los datos disponibles en la Web pudieran ser procesados de forma automática. La idea era que la web de entonces, formada por documentos, se convirtiera en una web formada por datos con significado y que las máquinas pudieran procesar toda esa información.

Con esa idea se desarrollaron toda una serie de tecnologías (RDF, OWL, SPARQL, GDDRL), pero con el paso del tiempo el interés por la web semántica ha ido decayendo. En diciembre de 2013 los grupos de trabajo dedicados a la web semántica se integraron en la actividad del W3C Data, pero simplemente como grupo de interés (es decir, sin el objetivo de producir recomendaciones).

Referencias:

  • Artículo La Web Semántica, de Berners-Lee en Investigación y Ciencia (marzo 2001)
  • Artículo Metacrap, de Cory Doctorow (agosto 2001), exponiendo las dudas sobre la posiblidad de construir la web semántica
  • 16th International Semantic Web Conference (ISWC 2017, Viena, octubre de 2017)
