La Web Semántica

La Web Semántica fue una iniciativa del W3C creada en 2001 con el objetivo de que los datos disponibles en la Web pudieran ser procesados de forma automática. La idea era que la web de entonces, formada por documentos, se convirtiera en una web formada por datos con significado y que las máquinas pudieran procesar toda esa información.

Con esa idea se desarrollaron toda una serie de tecnologías (RDF, OWL, SPARQL, GDDRL, etc.) que permitieran representar, codificar y consultar la información de forma automática.

La imagen siguiente muestra las fechas de aprobación de las recomendaciones relacionadas con las principales tecnologías de la web semántica (menos la última línea que muestra tecnologías menos importantes).

Desgraciadamente, con el paso del tiempo el interés por la web semántica ha ido decayendo. Las tecnologías desarrolladas para la web semántica se emplean en muchos sectores y se han integrado en el arsenal de técnicas que se engloban bajo el término Inteligencia Artifical, pero no tienen el papel central que esperaba el W3C.

En diciembre de 2013 los grupos de trabajo dedicados a la web semántica se integraron en la actividad del W3C Data, pero simplemente como grupo de interés (es decir, sin el objetivo de producir recomendaciones).

A posteriori, es fácil reconocer que el objetivo de la web semántica era demasiado ambicioso y que las herramientas propuestas no eran suficientes para poder conseguir su objetivo. Pero tampoco podemos olvidar que la base de la web semántica era el XML y por lo menos desde 2003 la web realmente existente se ha ido alejando cada vez más del XML, lo que complicaba en la práctica todo el proyecto.

Resource Description Framework (RDF)

RDF es un modelo de datos para metadatos, diseñado para representar la información disponible en la web. Las expresiones RDF tienen la forma sujeto-predicado-objeto (por eso se llaman tripletes).

Primer grupo de trabajo RDF en el W3C: Homepage - Charter: 2001-2002 - 2002-2004

Segundo grupo de trabajo RDF en el W3C: Homepage - Charter: 2011-2014

El primer grupo de trabajo RDF se debió crear en 1997 y se cerró en mayo de 2004. El segundo grupo de trabajo RDF se creó en febrero de 2011 y se cerró en julio de 2014.

El grupo de trabajo RDF publicó las siguientes recomendaciones:

  • febrero de 2014: RDF 1.1 Concepts and Abstract Syntax, editada por Richard Cyganiak, David Wood y Markus Lanthaler
  • febrero de 2014: RDF 1.1 Semantics, editada por Patrick J. Hayes y Peter F. Patel-Schneider
  • enero de 2014: JSON-LD 1.0 (A JSON-based Serialization for Linked Data), editada por Manu Sporny y otros
  • enero de 2014: JSON-LD 1.0 Processing Algorithms and API, editada por Dave Longley y otros
  • febrero de 2014: RDF 1.1 Turtle (Terse RDF Triple Language), editada por Eric Prud'hommeaux y Gavin Carothers
  • febrero de 2014: RDF 1.1 TriG (RDF Dataset Language), editada por Chris Bizer y Richard Cyganiak
  • febrero de 2014: RDF 1.1 N-Triples (A line-based syntax for an RDF graph), editada por Gavin Carothers y Andy Seaborne
  • febrero de 2014: RDF 1.1 N-Quads (A line-based syntax for RDF datasets), editada por Gavin Carothers
  • febrero de 2014: RDF 1.1 XML Syntax, editada por Fabien Gandon y Guus Schreiber
  • febrero de 2014: RDF Schema 1.1, editada por Dan Brickley y R.V. Guha

Web Ontology Language (OWL)

OWL es un conjunto de lenguajes para definir ontologías. Una ontología es un conjunto de términos que permiten referirse a unos objetos y a las relaciones entre ellos.
