En este página se listan las versiones obsoletas de las recomendaciones relacionadas con XML (XML, XSL, XQuery, etc.) y la web semántica (RDF, OWL, SPARQL, etc.). Las últimas versiones se comentan en las lecciones Normas y recomendaciones XML y Normas y recomendaciones Web semántica.

Recomendaciones XML obsoletas

XML Core

El grupo de trabajo XML Core publicó las siguientes recomendaciones que fueron sustuidas por versiones posteriores:

  • febrero de 1998: XML 1.0, editada por Tim Bray y otros.
  • enero de 1999: Namespaces in XML 1.0, editada por Tim Bray y otros.
  • junio de 1999: Associating Style Sheets with XML documents 1.0, editada por James Clark.
  • octubre de 2000: XML 1.0 (2º edición), editada por Tim Bray y otros.
  • marzo de 2001: Canonical XML 1.0, editada por John Boyer.
  • junio de 2001: XML Base, editada por Jonathan Marsh.
  • junio de 2001: XML Linking Language (XLink) 1.0, editada por Steve DeRose y otros.
  • octubre de 2001: XML Information Set, editada por John Cowan y Richard Tobin.
  • febrero de 2004: XML 1.0 (3º edición), editada por Tim Bray y otros.
  • febrero de 2004: XML 1.1, editada por Tim Bray y otros.
  • febrero de 2004: Namespaces in XML 1.1, editada por Tim Bray y otros.
  • diciembre de 2004: XML Inclusions (XInclude) 1.0, editada por Jonathan Marsh y David Orchard.
  • agosto de 2006: XML 1.0 (4º edición) editada por Tim Bray y otros.
  • agosto de 2006: Namespaces in XML 1.0 (2º edición), editada por Tim Bray y otros.
  • abril de 2010: Associating Schemas with XML documents 1.0, editada por Paul Grosso y Jirka Kosek.
  • agosto de 2011: Associating Schemas with XML documents 1.0 (2º edición), editada por Paul Grosso y Jirka Kosek.

El grupo de trabajo XML Core publicó las siguientes notas del grupo de trabajo:

  • octubre de 2003: LBase: Semantics for Languages of the Semantic Web, editada por R.V.Guha y Patrick Hayes
  • octubre de 2012: Associating Schemas with XML documents 1.0 (3º edición), editada por Paul Grosso y Jirka Kosek
  • julio de 2016: XML Inclusions (XInclude) 1.1, editada por Jonathan Marsh y otros

    Esta nota del grupo de trabajo es idéntica a la norma ISO/IEC 19757-11. El subcomité ISO/IEC JTC 1/SC 34 se encarga de la norma ISO 19757 (DSDL: Document Schema Definition Languages) que trata de varios lenguajes de definición de documentos (DTD, Schema, RELAX NG, Schematron, etc.). Algunas partes de esta norma se encuentran disponibles.

Al cerrarse el grupo de trabajo XML Core, varios proyectos de recomendaciones quedaron abandonados y ni siquiera se publicaron como notas del grupo de trabajo:

  • enero de 2010: XML Schema Definition Language (XSD): Component Designators
  • marzo de 2013: XML Normalization
