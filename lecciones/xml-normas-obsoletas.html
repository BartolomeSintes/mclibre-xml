En este página se listan las versiones obsoletas de las recomendaciones relacionadas con XML (XML, XSL, XQuery, etc.) y la web semántica (RDF, OWL, SPARQL, etc.). Las versiones actuales se comentan en las lecciones Normas y recomendaciones XML y Normas y recomendaciones Web semántica.
