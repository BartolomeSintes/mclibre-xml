Editores XML

Otros editores libres que podría utilizar en lugar de XML Copy Editor

  • Microsoft XML Notepad: https://microsoft.github.io/XmlNotepad/ - Rpositorio en GitHub - Descarga
  • Notepad++ con extensión XML Tools: http://notepad-plus-plus.org/ - Repositorio en GitHub de XML Tools

Otros editores que parece que ya no están en desarrollo:

  • CAM Editor: http://sourceforge.net/projects/camprocessor/ - Descarga - última versión: 2015 (algunos elementos actualizados con posterioridad)
  • XMLSpear: http://www.donkeydevelopment.com/ - última versión: 2016
  • Rinzo XML Editor: http://editorxml.sourceforge.net/index.html - Decarga - última versión: 2014
  • Jaxe: http://jaxe.sourceforge.net/en/ - Descarga - última versión: 2011
  • XPontus: http://xpontus.sourceforge.net/screenshots.html - Descarga - última versión: 2008
